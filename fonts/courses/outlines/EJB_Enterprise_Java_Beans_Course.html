ENTERPRISE JAVABEANS
2 Days
 
  PREREQUISITES: Students should have completed an advanced Java course or have equivalent knowledge. In particular, familiarity with RMI, JDBC, and Servlets is assumed.

WHO SHOULD ATTEND?: System architects and application programmers interested in developing server-side components and/or Web applications. The course is aimed towards Java developers with no prior experience with Enterprise Java Beans.

COURSE DESCRIPTION: The course is an overview of Enterprise Java Beans and related J2EE technologies. The EJB architecture is introduced along with some sample Beans, with an emphasis on the step-by-step process of designing, coding, and deploying the Bean.

BENEFITS OF ATTENDANCE: Students will be able to:

 
        Describe the characteristics of enterprise computing and multi-tiered applications;
        Understand relationships between EJB's and RMI;
        Set up an EJB server to develop, verify and test EJB's;
        Use a deployment tool to configure, install and maintain EJB's on a server;
        Write the interfaces and classes required for Enterprise Java Beans;
        Run client applications for EJB's deployed on a server;
        Develop web components for EJB application;
        Use JDBC for bean-managed persistence;
        Take advantage of container-managed persistence;
        Use facilities of the EJB server to manage transactions and security
